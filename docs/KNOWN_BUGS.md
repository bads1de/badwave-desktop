# 既知のバグ (Known Bugs)

現在確認されている、修正予定の不具合リストです。

## オフライン環境での再生に関する不具合

### 1. 「いいね」ページからの再生が一時停止状態で開始される

- **現象**: オフライン状態で「お気に入り（Liked Songs）」ページの楽曲をクリックした際、プレイヤーに楽曲情報は読み込まれるが、自動再生が開始されず一時停止（Paused）状態のままになる。
- **再現手順**:
  1. アプリをオフライン状態にする。
  2. 「いいねした曲」ページに移動する。
  3. リスト内の楽曲をクリックする。
- **期待される挙動**: クリックと同時に再生が開始されるべき。
- **現在の状況**: プレイヤーに表示されるだけで、再生ボタンを手動で押すまで再生されない。
- **推測される原因**:
  - `useOnPlay` フック、または `usePlayer` ストア内での再生開始処理が、オフライン時の状態遷移（isLoading や fetchStatus）と競合している可能性がある。
  - オーディオソース（URL）の解決と再生コマンドのタイミングの問題。
